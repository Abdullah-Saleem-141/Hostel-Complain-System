NOOR HOSTELS - HOSTEL COMPLAINT PORTAL
OWNER'S HANDOVER & SETUP GUIDE
===========================================================

This project is a professional web-based system for managing 
hostel complaints. It is connected to a secure cloud database
(Supabase) with high-performance connection pooling.

-----------------------------------------------------------
1. SYSTEM REQUIREMENTS
-----------------------------------------------------------
- Windows 10 or 11
- Java JDK 17 or higher installed 

-----------------------------------------------------------
2. HOW TO START THE SYSTEM
-----------------------------------------------------------
1. Open the project folder (Complain portal).
2. Right-click on the file named "start.ps1".
3. Select "Run with PowerShell".
4. Wait for the window to say: 
   'Starting ProtocolHandler [http-bio-8080]'
5. Open your web browser and go to: 
   http://localhost:8080

-----------------------------------------------------------

-----------------------------------------------------------
4. KEY FEATURES FOR OWNER
-----------------------------------------------------------
- Super Admin Feed: "Noor Hostels" account can see complaints from ALL hostels.
- Instant Updates: Mark issues as resolved using AJAX without page reloads.
- Visual Timeline: Track complaint progress from "Sent" to "Finished".
- Performance: Powered by HikariCP for fast data handling.
- Modern UI: Attractive animations and glassmorphism design.

-----------------------------------------------------------
5. PUBLIC ACCESS & QR CODE
-----------------------------------------------------------
To let students use this from their phones:
1. Host this folder on a cloud service.
2. Get the public URL.
3. Print the QR code and place it in the hostel halls.
